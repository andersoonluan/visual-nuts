## Installation

You can install using the latest nodejs version and latest npm version.

```bash
git clone https://github.com/andersoonluan/visual-nuts.git
cd visual-nuts
npm install
```

## Unit Test
We have unit test available using [Mocha](https://mochajs.org/)
```javascript
npm run test
```

## Run project
```javascript
npm run start
```
